# DataOS 架构文档

> 对标 Alibaba DataWorks + ByteDance DataLeap  
> 最后更新: 2026-06-06

---

## 一、项目定位

DataOS 是企业级数据治理平台，作为 **RelOS（关系操作系统）的上游**，负责多源数据采集、清洗、标准化和 API 服务化。RelOS 在其下游，接收标准化数据后进行关系图存储与推理。

```
DataOS（数据治理） → RelOS（关系推理） → AgentNexus（自然语言）
    上游                 中游               下游
```

                └──→  Nexus Ops (L1)

                      (MVP 阶段 Shadow Mode)
